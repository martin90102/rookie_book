## 简介：

​    这是一个为了新生培训的写的演示项目，主要用于帮助初学者理解一些图像处理的方法，掌握基本的编程概念和技巧。

## 内容：

​    大致会分为三个部分：图像分割、图像去噪声、特征点检测与匹配（时间充裕还会继续写）
​    在对应的文件夹下有各个部分的示例代码

## 编译：

```
    git clone https://github.com/martin90102/rookie_book.git
    mkdir build
    cmake ..
    make -j
```


用法：

```
    cd bin
    ./** ../images/lena.jpeg
```
